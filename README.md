# PowerPlan Support

A static website for PowerPlan app containing privacy policy, terms & liability information, and support contact details.

## Deployment

This site is deployed using GitHub Pages. The deployment is automated through GitHub Actions.

### Setup GitHub Pages

1. Go to your repository settings
2. Navigate to "Pages" under "Code and automation"
3. Under "Build and deployment", select "GitHub Actions" as the source
4. The site will automatically deploy when you push to the `main` branch

### Manual Deployment

You can also manually trigger a deployment from the Actions tab in your repository.

## Local Development

Since this is a simple static HTML site, you can open `index.html` directly in your browser to preview the site locally.
